### 배포용 버전 만들기

```shell
# 배포판 만들기
$ yarn run build
```

build 디렉토리가 생긴다.

serve 명령어를 설치한다.

```shell
$ npm install -g serve
```

serve 를 실행한다.

```shell
$ serve -s build
```
